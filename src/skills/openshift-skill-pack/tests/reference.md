# OpenShift Tests Reference

## Test Categories

| Category | Description |
|----------|-------------|
| Detection | Test detection rules for accuracy |
| Reasoning | Test engineering reasoning patterns |
| Workflow | Test workflow selection and steps |
| Command | Test command knowledge and explanations |
| Context | Test context interpretation |
| Documentation | Test documentation references |
| Safety | Test safety rules and warnings |

## Test Framework

Tests validate:
- Detection rules match expected patterns
- Reasoning patterns produce correct diagnoses
- Workflows select appropriate steps
- Commands provide accurate explanations
- Context interpretation is accurate
- Documentation references are valid
- Safety rules prevent dangerous recommendations

## Reference

Tests are implemented in the tests/ directory. See individual test files for details.
