# Nagios XI Workflows

## Overview

This directory contains detailed state-machine troubleshooting workflows for Nagios XI. Each workflow follows the STANDARD WORKFLOW MODEL.

## Workflow Index

1. [Service Monitoring Failure](service-monitoring-failure.md)
2. [Notification Failure](notification-failure.md)
3. [Configuration Apply Failure](configuration-apply-failure.md)

6. [Flapping Detection](flapping-detection.md)
7. [High CPU Usage](high-cpu-usage.md)
8. [Web Interface Issues](web-interface-issues.md)

## How to Use Workflows

1. Identify the observed symptom
2. Navigate to the corresponding workflow
3. Follow the structured investigation path
4. Collect evidence at each step
5. Apply recommendations with appropriate confidence scoring
6. Document findings and outcomes
