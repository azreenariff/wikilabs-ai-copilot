# EDB PostgreSQL Examples Reference

## Overview

This directory contains worked examples and reference implementations for EDB PostgreSQL.

## Example Categories

1. **Query Optimization**: EXPLAIN ANALYZE, index strategies
2. **Replication Setup**: Physical and logical replication
3. **High Availability**: Patroni, pgpool-II
4. **Backup and Recovery**: pg_dump, PITR
5. **Security Configuration**: SSL, authentication
6. **Performance Tuning**: Memory, WAL, I/O
