# MySQL Tests Reference

## Overview

This directory contains validation tests for the MySQL skill pack. Tests verify detection rules, knowledge retrieval, and workflow accuracy.

## Test Categories

1. Detection Rules Validation
2. Knowledge Retrieval Accuracy
3. Version-aware Knowledge Selection
4. Guidance Generation Quality
5. Workflow Selection Correctness
6. Cross-skill Consistency
7. Safety Rules Compliance
8. Confidence Scoring Validation

## Test Validation

Each test should verify:
- Detection rules match expected patterns
- Knowledge content is accurate
- Guidance follows safety standards
- Workflows follow the STANDARD WORKFLOW MODEL
- Confidence scoring is appropriate
- Version-aware guidance works correctly
