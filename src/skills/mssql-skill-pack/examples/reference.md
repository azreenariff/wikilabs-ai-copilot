# MSSQL Examples Reference

## Overview

This directory contains worked examples and reference implementations for Microsoft SQL Server.

## Example Categories

1. **Query Optimization**: Execution plans, index strategies
2. **Replication Setup**: Transactional, merge, peer-to-peer
3. **High Availability**: Always On, Log Shipping, Database Mirroring
4. **Backup and Recovery**: Full, differential, transaction log
5. **Security Configuration**: Authentication, encryption
6. **Performance Tuning**: Memory, I/O, CPU
